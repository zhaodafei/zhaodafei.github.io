命令格式

命令参数

在文件中搜索一个单词，命令会返回一个包含 PHP 的文本【行或者段落】

多个文件中搜索，命令会返回一个包含 PHP 的文本【行或者段落】

选项 -v 输出除 xxxxxx 之外的所有行或者段落

选项 -E 使用正则表达式

选项 -o 只输出文件中匹配到的部分

选项 -c 统计文本或者文本中包含匹配字符串的行数[ 选项 -n ]

选项 -l 搜索多个文件并查找匹配文本在那个文件中

选项 -r 递归搜索文件

选项 -i 忽略大小写搜索文件

选项 -e 扩展多个匹配样式

选项 -q 静默输出，一般配合 if 逻辑判断

选项 -A -B -C 显示匹配结果之后行、之前行、之前之后行
