    npm install hexo

使用项目提示–fei

新系统下载完这个项目后，node自行安装，开发使用的node版本为node-8.17.0 备用为node-v10.13.0-x64.msi

注意

.deploy_git文件中第一次使用会不识别大小写,这是由于Git默认对于文件名大小写是不敏感的

配置git使其对文件大小写敏感

然后你需要把远程中大小写的文件删除,然后把本地修改好的重新添加提交推送到远程(切记:直接修改后推送是没有效果的)
