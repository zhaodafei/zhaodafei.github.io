下载war 包

第一种安装方式：

前提：系统中必须安装好jdk，我的 jdk 是1.8

将下载好的war包放到linux中；切换到 jenkins.war　存放目录，输入命令： java -jar jenkins.war　
然后在浏览器中输入localhost:8080,就可以打开Jenkins

这种方式安装好后命令窗口会一直保持打开状态，不太实用；
