基本快捷键

  1. Ctrl+Shift+J：将选中的行合并成一行 (也可以把多行注释前面的*去掉)

  2. ctrl + 回车： 自动添加连接符换行

  3. Ctrl+G：定位到文件某一行

  4. Ctrl+D:复制当前行

  5. Ctrl+Y：删除当前行

  6. Ctrl+Alt+L：格式化代码

  7. Ctrl+Shift+U：切换大小写

  8. Ctrl+/：使用//注释

  9. Ctrl+Shift+/：使用/**/注释\

  10. Alt+Enter：自动提示完成，抛出异常

  11. Ctrl+F：在当前文件中查找

  12. Ctrl+R：替换字符串

  13. Ctrl+Shift+F:在全局文件中查找字符串

  14. Ctrl+Shift+R：在全局中替换字符串

  15. Ctrl+Z：撤销

  16. Ctrl+Shift+Z：反撤销

  17. 在项目文件上右键Compare With,对比文件

    1. 同时选中一个,在后面对话框中可以选择 项目外文 件对比
    2. 同时选中多哥,可以直接对比

  18. 数字书签

    数字书签

    ctrl + shift + 数字 标记后,可以用 ctrl + 数字 快速回到标记地方

  19. Java 开发中局部变量改为公用 Ctrl + Alt + f
