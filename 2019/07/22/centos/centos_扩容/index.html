在虚拟机VMware上增加磁盘空间

使用了一段时间虚拟机后发现磁盘不够用了,需要扩容。在客户端操作扩容出现磁盘已成功扩展。您必须从客户机操作系统内部对磁盘重新进行分区和扩展文件系统。 然后进centos7系统进行操作

使用 df -h 查看磁盘空间大小

使用 fdisk -l 命令查看磁盘信息

使用 fdisk /dev/sda 创建新的分区

重启系统

查看磁盘情况

使用 pvcreate /dev/sda4 创建物理卷

使用pvdisplay 查看新建的物理卷大小

使用 vgdisplay 查看 vg 状态

使用 vgextend centos /dev/sda4 将添加新的物理卷加载到 centos 卷组

使用 lvdisplay 查看 lv 状态

使用 lvextend 增加容量

查看当前分区类型(在下一步会用到)

使用 xfs_growfs (centos7)重新调整文件系统大小(根据上一步类型使用不同命令)
